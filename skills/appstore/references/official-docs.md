# 苹果官方文档索引（数值争议时的最终裁决源）

> 本 skill 各分册里的规格数字是写作时从官方核实的**速查缓存**。涉及尺寸/字数/时长等硬性规格且事关提交成败时，先 WebFetch 对应官方页核对再执行。链接失效（苹果常改版）就按标题 WebSearch 找新址，并顺手更新本文件。
> 本页只列高频页；**ASC 帮助全站 263 页的完整索引见 official-docs-sitemap.md**（按主题分 37 组，任何冷门主题先去那里定位）。
>
> **三个知识体系的分工与信任等级**：ASC Help（本索引，官方，操作与规格）｜App Review Guidelines（官方，审核条款原文）｜preflight/（**第三方社区知识**，MIT 引入：检测方法与拒审判例是作者/社区经验，官方无从验证）。查"怎么填/多大尺寸"来这里，查"会不会被拒/怎么检测"去 preflight/；preflight 的结论当审计线索用，与官方条款冲突时以 Review Guidelines 原文为准。
